Flutter组件学习之Text

2023/6/1 Flutter witget学习

简介

text 用于显示单一格式文本字符串的组件，它可以多行显示，也可通过设置单行显示。可以设置style 属性改变文本样式，如果省略就会使用DefaultTextStyle的值。

常用属性

data

必填值 ，要显示的字符串

Text("Hello world.")

maxLines

控制文本最多显示几行

selectionColor

文本选择时 选择区域的颜色

style

设置文本样式

TextStyle

  • color 文本颜色
  • fontFamily 文本字体
  • fontSize 文字大小
  • fontWeight 文字自重
  • letterSpacing 字符间间隔
  • wordSpacing 文字间间隔
  • height 行高
  • decoration 文字装饰 如下划线
  • decorationColor 装饰颜色

富文本显示

可以同 Text.rich() 或 RichText() 包含多个TextSpan进行富文本设置，TextSpan可以设置不同的样式

TextSpan

text 自身显示的文本

children 附加的子TextSpan 列表，

style 文字样式

recognizer 文本点击事件
