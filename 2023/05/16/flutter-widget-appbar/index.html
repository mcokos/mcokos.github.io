Flutter 组件学习之AppBar

2023/5/16 Flutter witget学习

简介

AppBar是个Material Design witget, AppBar有一个toolbar和其他隐藏组件构成，如TabBar 和 FlexibleSpaceBar. AppBar一般会显示一个或多个操作按钮。AppBar 一般用在Scaffold.appBar 属性中被Scaffold包裹，在屏幕顶部固定高度显示， 如果要使用可收缩滚动的AppBar，可以选择SliverAppBar。

AppBar会显示组件ToolBar, loading、title、actions 在botoom组件上方，bottom一般使用TabBar

常用属性

title

这个属性是设置一个witget用于显示标题,例如：

title: const Text('AppBar Sample')

toolbarHeight

这是属性是设置AppBar toolbar部分高度。

centerTitle

这个属性是设置标题是否居中显示， 默认false

backgroundColor

设置AppBar的背景色

actions

设置显示在title 后面的 一组witget 集合，一般使用IconButton.

 actions: <Widget>[
            IconButton(
              icon: const Icon(Icons.add_alert),
              tooltip: 'Show Snackbar',
              onPressed: () {},
            ),
            IconButton(
                icon: const Icon(Icons.navigate_next),
                tooltip: 'Go to the next page',
                onPressed: () {}),
          ]

actionsIconTheme

用于设置actions 各个witget 的color 、size、opacity

actionsIconTheme:
              const IconThemeData(color: Colors.black, size: 30, opacity: 0.5),

leading

设置显示在title 前面的 witget 集合，一般使用IconButton.

leading: IconButton(
            icon: const Icon(Icons.add_alert),
            tooltip: 'Show Snackbar',
            onPressed: () {},
          )

iconTheme

设置Toolbar的icons witget 的 color 、size、opacity

iconTheme:
              const IconThemeData(color: Colors.orange, size: 30, opacity: 0.5),

titleTextStyle

设置标题的文字样式

titleTextStyle

设置toolbar的文字样式

shadowColor

设置Appbar 阴影颜色

elevation

设置Appbar z轴高度，可以用于显示阴影

scrolledUnderElevation

设置appbar 下方控件滚动时Elevation的值

scrolledUnderElevation: 4.0

notificationPredicate

当可滚动部件在appbar 内部时，通过设置这个属性控制监听 让阴影显现

notificationPredicate: (ScrollNotification notification) {
            return notification.depth == 1;
          }

全部属性可以通过查看官网文档学习
